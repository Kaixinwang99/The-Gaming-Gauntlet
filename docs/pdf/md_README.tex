\doxysubsection*{Introduction}

The Gaming Gauntlet is the new generation of videogame controllers that aims to show the world a new way for people interact with consoles.

\doxysubsection*{Installing}

\doxysubparagraph*{Configuration-\/\+Requirements}

For setting up the raspberry pi as a gaming controller it is necessary to create and deploy a report descriptor that tells the PC that it is an H\+ID \textquotesingle{}a controller\textquotesingle{}, for doing this we used the open-\/sourced milador raspberry\+\_\+py joystick for 8 buttons that can be found \href{https://github.com/milador/RaspberryPi-Joystick/tree/master/8_Buttons_Joystick}{\texttt{ here}}
